# 🐍 Python Project Portfolio

### 1. [📊 Body Analytics Utility](./bmi_analytics.py)
A refined tool designed to calculate Body Mass Index (BMI) with a focus on unit precision and inclusive data categorization.

**Technical Highlights:**
* **Standardized Logic:** Handles calculations across diverse measurement systems.
* **User Experience:** Features a recursive loop for a seamless, continuous experience.

---

### 2. [🧮 Arithmetic Logic Tool](./calculator.py)
A terminal-based calculator designed to perform fundamental arithmetic operations with robust error handling.

**Technical Highlights:**
* **Error Resilience:** Implements `try-except` blocks to manage non-numeric inputs and prevent "division by zero" crashes.
* **Control Flow:** Utilizes a `while` loop with a built-in termination command for continuous usage.
